4 meg Linux bootfile for USB memory sticks.
Creates a bootable memmory stick.
Ensure that your BIOS is set to boot off a memory stick in the first instance. This is the biggest headache to do.
Insert Memory stick.
Turn on computer, should boot the small Linux program from off your USB memory stick.
Follow the text menu on the screen to reset your admin and other users passwords. Typically select 'no password' initially.
